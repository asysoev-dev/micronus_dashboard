# Micronus - Умный дом

**Панель управления для системы умного дома** с настраиваемыми виджетами и дашбордами.

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Настройка окружения
cp .env.example .env
# Заполните DATABASE_URL, JWT_SECRETS

# Запуск в dev
npm run dev

# Сборка
npm run build
npm run start
```

## Требования

- Node.js 18+
- PostgreSQL 12+
- Nuxt 4
